# 城まもり！ 〜和風タワーディフェンス〜

スマホのブラウザで遊べる和風タワーディフェンスゲーム。侍・巫女などの守り手を配置して、鳥居から攻めてくる妖怪の大軍勢からお城を守ろう。

**▶ 遊ぶ: https://yasufabric.github.io/hirari/**

## 遊び方

1. マップを選んでスタート（クリアすると次のマップが解放される）
2. 下のパネルから守り手を選び、空いているマスをタップして配置
3. 「WAVE開始」をタップすると妖怪が攻めてくる
4. ゴールドを貯めて守り手を強化・増設し、全20ウェーブを守りきればクリア！

### 守り手

| 守り手 | 特徴 |
| --- | --- |
| 🏹 弓侍 | 連射が速い基本ユニット |
| ⛩️ 巫女 | お札で妖怪を清めて移動を遅くする |
| 💥 大筒 | 爆発で周囲の妖怪をまとめて攻撃 |
| 🎯 火縄銃 | 超射程・鎧貫通の一撃 |

### 妖怪

👻 おばけ ／ 👺 烏天狗（速い）／ 👹 鬼（硬い）／ 🦝 化けだぬき（回復する）／ 💀 落ち武者（鎧持ち）／ 🦊 九尾の狐（ボス）

## 開発

```bash
npm install
npm run dev        # http://localhost:8787
npm test           # ユニットテスト (Vitest)
npm run test:e2e   # スモークテスト (Playwright)
```

構成の詳細は `CLAUDE.md` を参照。
